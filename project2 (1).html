Tech Glossary

HTML
HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects such as interactive forms may be embedded into the rendered page. HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items.
CSS
CSS, or Cascading Style Sheets, is a stylesheet language used to define the visual presentation and formatting of web documents written in markup languages like HTML or XML. It works alongside HTML, which provides the structure and content, to control elements such as fonts, colors, layouts, spacing, and the overall aesthetic appearance of a website.
JS
JavaScript (JS) is a lightweight, interpreted, object-oriented programming language, primarily known as the scripting language for web pages. It enables interactive, dynamic content on the client-side (browsers) and powers backend development (Node.js). It is a core technology for web development alongside HTML and CSS.
